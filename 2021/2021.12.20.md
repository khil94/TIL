- Optional chaining operator
  - props로 받은 함수가 존재할 때, 실행시키는 방법은 다음과 같다.
    ```
      onClick && onClick();
    ```
  - 하지만 더욱 간결한 방법이있다.
    ```
      onClick?.();
    ```
    이는 Optional chaining 이라고 하는 연산자 이다.
